# EU Regulatory Seals

This directory contains the 15 reference EU regulatory seal images used for compliance validation.

## Instructions for Adding Real Seals

Replace the placeholder files with actual EU CLP/GHS regulatory seal images:

1. Name files descriptively (e.g., `explosive.png`, `flammable.png`, `corrosive.png`)
2. Use high-quality PNG images with transparent backgrounds
3. Ensure consistent sizing (recommended: 512x512px minimum)
4. Update the seal metadata in `sealData.ts` with correct descriptions

## Common EU CLP/GHS Pictograms

1. **Explosive** (`GHS-explosive`)
2. **Flammable** (`GHS-flammable`)
3. **Oxidising** (`GHS-oxidising`)
4. **Gas Under Pressure** (`GHS-gas-under-pressure`)
5. **Corrosive** (`GHS-corrosive`)
6. **Acute Toxicity** (`GHS-acute-toxicity`)
7. **Health Hazard** (`GHS-health-hazard`)
8. **Serious Health Hazard** (`GHS-serious-health-hazard`)
9. **Hazardous to the Environment** (`GHS-hazardous-to-environment`)
